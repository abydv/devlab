# DevLab REST API

No API endpoints exist yet. The REST API (Fiber, under `api/`) is
introduced in Sprint 11 — REST API, after the Engine, Workspace Manager,
Services, and Runtimes it delegates to have been implemented.

This document will be updated with endpoint definitions, request/response
schemas, and status codes as they are implemented. The API layer itself
will contain no business logic — it validates input and delegates to the
Engine.
